# OMR Bubble Sheet Detector

A Python-based OMR (Optical Mark Recognition) system using OpenCV.

## Features
- Detects OMR sheet
- Warps perspective
- Extracts answers automatically
- Supports multi-column bubble sheets

## Tech Stack
- Python
- OpenCV
- NumPy
